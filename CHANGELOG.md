# Changelog

本文件由 GitHub Actions 在每次合并到 `main` 时自动追加。
格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/)：倒序、按类型分组、每条关联 PR。

条目类型：

- `feat` 新功能
- `fix` 缺陷修复
- `docs` 文档
- `ci` 持续集成 / 构建
- `refactor` 重构
- `chore` 杂项
